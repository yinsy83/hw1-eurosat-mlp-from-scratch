# 从零开始构建三层神经网络分类器实现 EuroSAT 地表覆盖分类

## 1. 任务简介

本实验面向 EuroSAT RGB 遥感图像数据集，任务是使用从零实现的三层神经网络分类器对卫星图像进行地表覆盖分类。数据集共包含 10 个类别，分别为 `AnnualCrop`、`Forest`、`HerbaceousVegetation`、`Highway`、`Industrial`、`Pasture`、`PermanentCrop`、`Residential`、`River` 和 `SeaLake`。每张图像尺寸为 `64 x 64 x 3`。

本作业要求不能使用 PyTorch、TensorFlow、JAX 等具备自动微分能力的框架，因此本实验完全基于 `NumPy` 自主实现自动微分与反向传播机制，并在此基础上实现三层多层感知机分类器、SGD 优化器、学习率衰减、交叉熵损失、L2 正则化、验证集模型选择、超参数搜索和测试评估。

## 2. 数据处理

### 2.1 数据组织方式

数据集根目录 `EuroSAT_RGB` 下，每个类别对应一个子文件夹，文件夹中存放该类别的 RGB 图像。这种组织方式非常适合直接按照文件路径遍历并构造图像分类数据集。

### 2.2 数据预处理

本实验采用如下预处理流程：

- 读取图像并转换为 RGB 格式；
- 将像素值从 `uint8` 转换为 `float32`；
- 归一化到 `[0, 1]`；
- 将每张 `64 x 64 x 3` 图像展平为长度为 `12288` 的一维向量；
- 使用训练集统计得到的均值和标准差对训练集、验证集和测试集进行标准化。

为了避免数据泄漏，归一化参数只由训练集计算得到，然后复用于验证集与测试集。

### 2.3 数据划分

数据集采用分层划分策略：

- 训练集：70%
- 验证集：15%
- 测试集：15%

分层划分保证了每个类别在三个子集中的分布近似一致，从而使模型评估更加公平稳定。

## 3. 模型设计

### 3.1 三层 MLP 结构

本实验实现的三层神经网络采用如下结构：

```text
Input (12288)
  -> Linear(12288, H1)
  -> Activation
  -> Linear(H1, H2)
  -> Activation
  -> Linear(H2, 10)
  -> Logits
```

其中：

- `H1` 和 `H2` 为可调隐藏层维度；
- 激活函数支持 `ReLU`、`Sigmoid` 和 `Tanh`；
- 输出层维度固定为 10，对应 EuroSAT 的 10 个类别。

本实验主要比较如下隐藏层结构：

- `[256, 128]`
- `[512, 256]`

### 3.2 参数初始化

为了提升训练稳定性，线性层权重采用 Xavier 风格初始化。相较于过大的随机初始化，这种方式在 `ReLU` 和 `Tanh` 两类激活函数下都具有更好的数值稳定性，有利于防止训练早期出现梯度爆炸或 logits 溢出。

## 4. 自动微分与反向传播实现

### 4.1 自定义 Tensor

本实验没有使用现成自动微分框架，而是自行设计了 `Tensor` 类。每个 `Tensor` 对象保存以下信息：

- `data`：张量本身的数据；
- `grad`：反向传播得到的梯度；
- `parents`：当前节点在计算图中的父节点；
- `_backward`：当前节点的局部梯度传播函数；
- `requires_grad`：指示该张量是否需要参与梯度计算。

### 4.2 计算图与反向传播

在前向传播过程中，每一次运算都会生成新的 `Tensor` 节点，并记录当前节点与父节点之间的依赖关系。调用 `backward()` 时，程序会先对计算图做拓扑排序，再按照逆拓扑顺序逐层回传梯度，从而实现自动微分。

本实验实现了以下基础运算及其梯度：

- 加法、减法、乘法、除法；
- 矩阵乘法；
- 求和、求均值；
- reshape、transpose；
- ReLU、Sigmoid、Tanh；
- `exp` 与 `log`。

### 4.3 交叉熵损失

分类任务采用 Softmax Cross-Entropy Loss。为提高数值稳定性，先对 logits 做最大值平移，再计算 softmax 概率和负对数似然，从而避免指数运算中的数值上溢问题。

### 4.4 L2 正则化

为了抑制模型过拟合，本实验在损失函数中加入 L2 正则化项：

```text
Loss = CrossEntropy + 0.5 * lambda * ||W||^2
```

其中 `lambda` 表示正则化强度。

## 5. 优化方法

### 5.1 SGD

参数更新采用最基础的随机梯度下降（SGD）方法：

```text
theta = theta - lr * grad
```

### 5.2 学习率衰减

为了提升训练后期的收敛稳定性，本实验使用 Step Decay 学习率衰减策略：

```text
lr_epoch = lr_init * gamma ^ floor((epoch - 1) / step)
```

学习率在固定 epoch 间隔后按比例衰减。

### 5.3 最优模型保存

每轮训练结束后，在验证集上计算准确率。如果当前验证集准确率优于历史最优值，则自动保存当前模型参数，确保最终测试时使用的是验证集意义上的最优模型，而不是最后一个 epoch 的参数。

## 6. 超参数搜索

### 6.1 搜索设置

为了获得较优模型，本实验先进行稳定性试探，再对稳定区域内的配置做正式网格搜索。用于生成表格结果的正式搜索范围如下：

| 超参数 | 备选值 |
|---|---|
| learning rate | 0.01 |
| hidden dims | [256,128], [512,256] |
| weight decay | 1e-4, 1e-3 |
| activation | ReLU, Tanh |

搜索命令如下：

```bash
python src/search.py ^
  --dataset-root EuroSAT_RGB ^
  --epochs 4 ^
  --batch-size 64 ^
  --lrs 0.01 ^
  --weight-decays 0.0001,0.001 ^
  --activations relu,tanh ^
  --hidden-grid 256,128;512,256 ^
  --lr-decay-gamma 0.5 ^
  --lr-decay-step 2
```

在此之前，还额外对更大的学习率范围做过稳定性探测，发现 `0.01` 附近的设置收敛更稳定，因此正式比较阶段聚焦在该学习率附近展开。

### 6.2 搜索结果

正式搜索结果汇总如下：

| 排名 | activation | hidden dims | lr | weight decay | best val acc | best epoch |
|---|---|---|---|---|---|---|
| 1 | ReLU | [512, 256] | 0.01 | 1e-3 | 0.6105 | 3 |
| 2 | ReLU | [512, 256] | 0.01 | 1e-4 | 0.5992 | 4 |
| 3 | ReLU | [256, 128] | 0.01 | 1e-4 | 0.5945 | 3 |

从搜索结果可以看出：

- `ReLU` 明显优于 `Tanh`；
- 更大的隐藏层结构 `[512, 256]` 整体优于 `[256, 128]`；
- 适度增大正则化强度到 `1e-3` 在较大模型上有助于提升验证集表现。

因此，后续正式训练优先采用 `ReLU + [512, 256] + lr=0.01 + weight_decay=1e-3` 作为最终候选配置。

## 9. 第一层权重可视化分析

本实验将第一层隐藏层权重矩阵恢复为图像尺寸并进行可视化，用于观察网络是否学习到某些与颜色或空间纹理相关的模式。

第一层权重可视化如下：

![第一层权重可视化](../outputs/weight_viz/compare_relu_512_256_wd1e3_first_layer_weights.png)

从结果中通常可以观察到以下现象：

- 部分隐藏单元对绿色纹理响应较强，可能与森林或植被类相关；
- 部分隐藏单元对蓝色或高亮区域更敏感，可能与河流、湖泊等水体类别有关；
- 一些单元表现出条带状或边缘状结构，可能更偏向提取道路、河流等细长空间模式；
- 也有一些单元更像是在编码整体颜色分布，而不是明确的局部形状。

这些现象说明，即使是简单的 MLP，在像素级输入下也能够学习到一定的颜色和空间统计模式，但其对局部几何结构的建模能力仍然有限。

## 10. 错例分析

测试集中的错误分类样本如下：

![错例分析](../outputs/error_cases/compare_relu_512_256_wd1e3_error_cases.png)

从典型错例中，可以进行如下分析：

1. `River` 与 `Highway` 可能被混淆，因为两者都可能呈现细长带状结构。
2. `Pasture` 与 `HerbaceousVegetation` 颜色和纹理分布相近，因此容易互相误分。
3. `Residential` 与 `Industrial` 都包含大量人工建筑区域，整体纹理分布可能接近。
4. 对于场景中包含多种地物的样本，单一主类别标签本身就会增加分类难度。

总体来看，这些错例多与类别间的视觉相似性和场景混杂有关，也反映了 MLP 缺乏局部卷积建模能力的局限性。

## 11. 总结

本实验基于 `NumPy` 从零实现了自动微分与三层 MLP 分类器，并在 EuroSAT 数据集上完成了土地覆盖分类任务。实验中实现了数据预处理、模型定义、训练循环、验证集模型选择、测试集评估和超参数搜索等完整流程，满足了课程作业对自主实现深度学习基础模块的要求。最终选定的 `ReLU + [512, 256]` 配置在验证集上达到 `0.6582` 的最佳准确率，在测试集上达到 `0.6614` 的准确率。

从实验结果可以看出，三层 MLP 在该任务上能够学习到有意义的颜色和纹理模式，并取得一定分类精度；同时，通过权重可视化和错例分析可以看出，模型仍然受限于其对局部空间结构建模能力较弱的问题。未来若允许使用更复杂结构，可以考虑卷积网络以进一步提升性能。
